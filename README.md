# rails-docker-mysql

# Overview
This is how to build an environment to run rails and mysql using Docker.

# Requirement
- Rails 6.1.7
- ruby 3.0.5p211
- mysql 8.0.31

* Start the container

```

```

* Check on website

[Check](http://localhost:3000/)
